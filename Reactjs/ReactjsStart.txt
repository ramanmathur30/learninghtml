1.) Create a folder - Rename your project name
2.) In your project folder- Open cmd, set path in cmd of your project folder path
3.) Enter command in cmd:-(To create package.JSON=To install node_module or libraries)
	npm init -y
	or
	npm init

- package.json
	-It is generated by cmd-> npm init  

- main 
	- Project start from main
- dependencies 
	- All node_module "file name" with "version" description

- scripts
	- It is used to write scripts in it for START project to perform html page.
